## Built with

- ReactJs
- Css

## Run Locally

Clone the project

```bash
  git clone https://github.com/shifachaus/usePopcorn.git
```

Go to the project directory

```bash
  cd usePorpcorn
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
